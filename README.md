# greenhouse



Materiales del invernadero: policarbonato
El policarbonato se puede curvar ligeramente alrededor del marco y permite un ahorro de energía de hasta 30%, porque tiene dos capas.
A través de él se filtra el 80% de la luz.

Calefactor electronico


Motores del cohete
    El tanque superior de oxígeno líquido (LOX).
    El tanque intermedio no presurizado que contiene la mayoría de los componentes eléctricos.
    El tanque inferior de hidrógeno líquido (LH2); es la parte más larga, aunque es relativamente ligera.
